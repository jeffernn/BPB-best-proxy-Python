1.BPB面板.py                                  
功能为从fofo中下载的csv文件读取有效信息，套入漏洞格式后转为txt。                                                                                       
2.BPB筛选域名.py                                                                                                                                                          
配合BPB面板.py筛选所需要的域名后缀。                                  
3.BPB面板节点自动上传.py                                                                                                                                                          
将筛选的域名自动上传到聚合订阅地址。                                  
4.BPB全自动.py                                                                                   
结合以上1，2，3软件的功能全自动化。                                  
5.从工具中获取优选IP导出为txt.py                                  
配合cloudflare筛选优选ip导出的csv文件，提取有效，速度>0的ip地址导出为txt文档。                                  
6.从网站中获取优选IP导出为txt.py                                  
从自动更新的优选ip网站中筛选优选ip导出的csv文件，提取有效，速度>0的ip地址导出为txt文档。
